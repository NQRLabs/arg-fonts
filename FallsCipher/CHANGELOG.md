# Changelog - Falls Cipher Font

All notable changes to the Falls Cipher font will be documented in this file.

## [1.0.0] - 2025-01-11

### Added
- Initial release of Falls Cipher font (formerly Gravityfalls)
- Complete A-Z alphabet mapping to mystery-themed symbol cipher
- Unique glyph design for each letter
- TTF and OTF formats
- WOFF2 format for web use
- Thematic symbols combining geometric and occult aesthetics
- First in planned Falls Cipher series

### Font Specifications
- **Glyphs:** 26 letters (A-Z)
- **Weight:** Regular
- **Style:** Cipher/Symbolic/Mystery
- **Formats:** TTF, OTF, WOFF2
- **Series:** Falls Cipher #1

### Notes
- Renamed from "Gravityfalls" to "Falls Cipher" to avoid trademark issues
- Designed to accommodate future numbered releases (FallsCipher-02, etc.)
- Based on symbol substitution cipher from mystery-themed media
