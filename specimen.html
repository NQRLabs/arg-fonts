How to Use

  • Upload your font file - TTF, OTF, WOFF, or WOFF2 format
  • Edit the font name - This appears prominently in the specimen
  • Choose background color - Default matches NQR Labs theme (#04070a)
  • Preview updates automatically - See changes in real-time
  • Download as PNG - Saves as [fontname]-specimen.png (1200x400px)

Specimen Contents

  • Font name - Large, prominent display (96px)
  • Row 1: Uppercase alphabet (A-Z) - Full width
  • Row 2: Lowercase alphabet (a-z) - Full width
  • Row 3: Numbers (0-9) and Special characters side-by-side
  • Special characters - `~!@#$%^&*()-_=+[{]}\|;:'",<.>/?
  • Layout: Three-row design for consistent single-line alphabets
