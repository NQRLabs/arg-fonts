# Changelog - NQR Labs ARG Cipher Fonts Pack

All notable changes to the ARG Cipher Fonts collection will be documented in this file.

## [1.0.0] - 2025-11-11

### Initial Release

This is the first public release of the NQR Labs ARG Cipher Fonts pack, containing three handcrafted cipher fonts designed for puzzle creators, ARG designers, and cryptography enthusiasts.

### Fonts Included

#### Zodiac Cipher v1.0.0
- Complete cipher font based on Zodiac Killer symbols
- 26 letters + numbers + punctuation
- Historically accurate symbol mappings
- **[View Details](ZodiacCipher/CHANGELOG.md)**

#### Pigpen Cipher v1.0.0
- Classic Freemason/Masonic geometric cipher
- Traditional grid-based symbol system
- 26 letters (A-Z)
- **[View Details](PigpenCipher/CHANGELOG.md)**

#### Falls Cipher v1.0.0
- Mystery-themed symbol substitution cipher
- Inspired by cryptographic puzzles in adventure media
- First in planned Falls Cipher series
- **[View Details](FallsCipher/CHANGELOG.md)**

### Package Features

- **Multiple formats** - TTF, OTF, and WOFF2 for each font
- **Web-ready CSS** - Pre-built @font-face declarations
- **Complete documentation** - README, specimen images, character maps
- **Open source** - Licensed under SIL Open Font License 1.1
- **Created with Calligraphr** - Professional web-based font creation tool

### Documentation

- Main README with gallery and usage guide
- Per-font documentation with specimens and character maps
- Quick-start PACK-README for downloads
- CSS files for easy web integration
- OFL license files for each font

## Version Numbering

This pack uses [Semantic Versioning](https://semver.org/):
- **MAJOR** - New fonts added or breaking changes
- **MINOR** - New features, character additions, or improvements
- **PATCH** - Bug fixes and minor adjustments

Individual fonts have their own version numbers documented in their respective changelogs.
