# NQR Labs ARG Cipher Fonts

A collection of handcrafted cipher fonts for puzzle creators, ARG designers, and cryptography enthusiasts.

## Download Fonts

Visit **https://nqrlabs.com/arg-fonts/** to view specimens and download individual fonts.

## What's Included

- **Zodiac Cipher** - Based on the Zodiac Killer's cipher symbols
- **Pigpen Cipher** - Classic Freemason/Masonic grid cipher
- **Falls Cipher** - Mystery-themed symbol substitution cipher

## Formats

Each font is available in:
- **TTF** - TrueType Font (Windows, Linux)
- **OTF** - OpenType Font (macOS, cross-platform)
- **WOFF2** - Web Open Font Format 2 (optimized for web use)

## Quick Start

### Desktop Installation
1. Download the font file for your system (TTF for Windows/Linux, OTF for macOS)
2. Double-click and install
3. Use in any application

### Web Usage
1. Include the font files on your server
2. Copy the `@font-face` CSS from each font's `css/fontface.css`
3. Apply the font in your stylesheets
