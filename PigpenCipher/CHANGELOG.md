# Changelog - Pigpen Cipher Font

All notable changes to the Pigpen Cipher font will be documented in this file.

## [1.0.0] - 2025-01-11

### Added
- Initial release of Pigpen Cipher font
- Complete A-Z alphabet mapping to traditional Pigpen cipher symbols
- Grid-based symbol system (3x3 grid + X grid + dots)
- TTF and OTF formats
- WOFF2 format for web use
- Clean geometric glyph design for maximum legibility
- Consistent line weight across all symbols

### Font Specifications
- **Glyphs:** 26 letters (A-Z)
- **Weight:** Regular
- **Style:** Cipher/Geometric
- **Formats:** TTF, OTF, WOFF2
- **Grid System:** Traditional Pigpen (3x3 + X + dots)
