# Changelog - Zodiac Cipher Font

All notable changes to the Zodiac Cipher font will be documented in this file.

## [1.0.0] - 2025-01-11

### Added
- Initial release of Zodiac Cipher font
- Complete A-Z alphabet mapping to Zodiac cipher symbols
- Basic punctuation and number support
- TTF and OTF formats
- WOFF2 format for web use
- Character mapping based on authentic Zodiac cipher symbols
- Optimized kerning for readability

### Font Specifications
- **Glyphs:** 26 letters + numbers + punctuation
- **Weight:** Regular
- **Style:** Cipher/Symbolic
- **Formats:** TTF, OTF, WOFF2
